# Git

# 推荐资源

  • GitBook (opens new window)：有中文版，且较权威，更偏理论
  • Git Apprentice (opens new window)：全英书籍，提供较多有趣的实例，学习过程会更加有趣
  • Advanced Git (opens new window)：同上，是上述资源的提升版

# 学习要求

  • 熟练掌握常用命令(提交、分支、远端同步、冲突解决等)，工作流(Gitflow工作流、基于功能分支工作流等)，对基本概念(提交、分支、标签、冲突等)熟练掌握
  • 对于其内部的工作原理可以不熟记，但要求了解，需要知道一些常用命令执行时发生了什么
